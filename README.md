# Caça-Palavras Agro Sustentável

## Sobre o projeto

O **Caça-Palavras Agro Sustentável** é um jogo educativo sobre agro, sustentabilidade e meio ambiente.

O jogador precisa encontrar palavras escondidas na grade. As palavras podem aparecer na horizontal, vertical ou diagonal.

Tema:

**Agro forte, futuro sustentável: equilíbrio entre produção e meio ambiente**

## Funcionalidades

- Caça-palavras jogável.
- Seleção de letras com clique.
- Palavras encontradas ficam marcadas.
- Pontuação automática.
- Cronômetro.
- Contador de palavras encontradas.
- Lista com 14 palavras.
- Layout responsivo.

## Como usar

1. Baixe o projeto.
2. Abra a pasta no VS Code.
3. Abra o arquivo `index.html` no navegador.
4. Clique nas letras em sequência para formar as palavras da lista.

## Objetivo educativo

Trabalhar vocabulário ligado ao campo, produção rural, sustentabilidade, preservação ambiental e uso consciente dos recursos naturais.
